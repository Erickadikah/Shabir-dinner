# Shabir-dinner

## Stacks used

- React
- Redux
- Next.js
- JavaScript
- Tailwind CSS
- Vercel
- NextAuth.js

## How to run

- Clone the repo
- Run `npm install`
- Run `npm run dev`
- Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
